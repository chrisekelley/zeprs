1.   Introduction

The ZEPRS application provides electronic access to perinatal medical records, for pregnant women and their newborns up to 6 weeks.  This is a web-based application, and the goal is to initially use this in parallel with the blue books, in the Lusaka health district and UTH, until such time as the system is fully tested, and all medical staff are trained in using ZEPRS.

 

Note, not all of the features listed here are fully implemented yet.  In the instance where the screen shot looks different from what will be implemented, there is an [UPDATE] note which lists the proposed changes.  [UPDATE] is in red for the beta release, and in pink [UPDATE] for the full Version 1.0 release.

 

Also, the enumerations are not listed in this User Manual, but you are referred to the attached Medical Records spreadsheet which includes all updates for forms with their enumerations.  This User Manual does list however where elements on the forms have been added or deleted.
